Bienvenido/a a Recetapp

Recetapp nació de la necesidad de poder ahorrar tiempo para poder hacer la lista de compras para la comida. También de la idea de poder hacer mejores recetas, más ricas, ahorrando tiempo y dinero. ¡Guten appetit!

Recetapp es un proyecto hecho con Django

Django de un framework web escrito en Python. Recetapp es un proyecto para poder practicar django y también para poder gestionar nuestras recetas y compras.
